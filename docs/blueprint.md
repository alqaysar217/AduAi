# **App Name**: EduAI Simplified

## Core Features:

- Simplified Onboarding: Implement a streamlined onboarding flow with clear welcome screens and easy login/signup options.
- Consistent UI Design: Develop a consistent UI across all screens, adhering to the specified color palette, typography, and UI components.

## Style Guidelines:

- Primary color: #7e57c2 (Purple) to establish a consistent brand identity.
- Use #4caf50 (Success), #ffc107 (Warning), and #ff5252 (Error) for feedback and alerts.
- Accent: Teal (#008080) for interactive elements and highlights.
- Maintain consistent typography using 'Roboto Bold' for headings and 'Roboto Regular' at 16px for body text.
- Employ cards with 'Shadow: 4px 8px 0 rgba(0,0,0,0.10)' and '16px rounded corners' to organize content effectively.

## Original User Request:
Design a mobile app called 'EduAI,' an AI-powered educational platform designed for both learners and educators. The app should have a user-friendly interface with a focus on easy navigation, smart interactions, and an engaging learning experience. Ensure consistency across 28 screens with the following design guidelines:

Colors:

    Primary: '#7e57c2' (Purple)

    Secondary: '#4caf50' (Success), '#ffc107' (Warning), '#ff5252' (Error)

    Text: '#333333' (Dark), '#666666' (Secondary)

Fonts:

    Headings: 'Roboto Bold' (Arabic: 'Tajawal Bold')

    Body Text: 'Roboto Regular' at '16px'

UI Components:

    Buttons: "12px rounded corners", "48px height"

    Input Fields: "Border: #e0e0e0", "Focus Shadow: #7e57c2"

    Cards: "Shadow: 4px 8px 0 rgba(0,0,0,0.10)", "16px rounded corners"

App Screens:

    App Launch Screen:

        Display a loading screen with the app's logo and name in the center, along with a brief description of the app below the logo.

    Welcome Screens (3 screens):

        First Screen: Display the app logo and the phrase "Welcome to EduAI" with a paragraph saying, "Start your journey with us to discover a new world of possibilities and amazing opportunities." At the bottom, include "Next" and "Login" buttons.

        Second Screen: Display the app logo and the phrase "Unique Features" with a paragraph saying, "Enjoy an exceptional user experience with advanced tools that help you achieve your goals." At the bottom, include "Next" and "Login" buttons.

        Third Screen: Display the app logo and the phrase "You're Ready" with a paragraph saying, "Log in now to start using the app and access all features." At the bottom, include "Sign Up" and "Login" buttons.

    Login Screen:

        Users should enter their username/email and password. Include a "Login" button, a "Remember Me" option, and a "Forgot Password" link leading to a reset password screen. Additionally, offer Google and Facebook login options. Include a link to "Sign Up" for users who don't have an account.

    Sign Up Screen:

        Users should enter their username, email, and password. Include a "Sign Up" button, Google and Facebook account creation options, and a link to "Login" for existing users."
